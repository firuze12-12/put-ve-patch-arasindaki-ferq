# PUT və PATCH Metodları Arasındakı Fərqlər

Bu repo müəllim tərəfindən verilən araşdırma tapşırığı üçün hazırlanıb.

## 1. PUT Metodu (Tam Yeniləmə)
- Məlumatın hamısını göndərməyi tələb edir.
- Mövcud məlumatı silir və göndərilən yeni məlumatla əvəz edir.
- **Nümunə:** Bir adamın adını, soyadını və yaşını dəyişmək üçün hamısını yenidən göndərmək.

## 2. PATCH Metodu (Qismən Yeniləmə)
- Yalnız dəyişən hissəni göndərmək kifayətdir.
- Mövcud məlumatın digər hissələrinə toxunmur.
- **Nümunə:** Adamın sadəcə soyadını dəyişmək üçün yalnız soyadı göndərmək.
